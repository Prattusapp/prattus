# Project Rules

- **Resource Usage**: The folder `antigravity-kit-main` is for reference/consumption only.
- **Development Location**: All new features, code, and project files must be created directly in the `Prattus APP` root or its subfolders (excluding `antigravity-kit-main`).

## Mandatory Workflow
1. **Consultation**: For every user request, first search `antigravity-kit-main/.agent/skills/` and `antigravity-kit-main/.agent/agents/` for relevant experts or skills.
2. **Justification**: Inform the user WHICH skills/agents will be used and WHY before proceeding with any implementation.
3. **Execution**: Implement the task in `Prattus APP` following the patterns identified in the consultation phase.

## Implementation Instructions
When asked to create new screens, components, or services, reference the skills and patterns in `antigravity-kit-main` but implement them in the main project structure.
